Outfit Rater (Python Project)
Description

This is a text-based Python application where the user creates an outfit by selecting a top, bottom, shoes, and a style. The program calculates a score based on these choices and returns both a rating and a style recommendation.

The purpose of this project is to demonstrate fundamental programming skills equivalent to a beginner-level programming course.

Features
User input via terminal
Score calculation based on selected items
Rating system depending on total score
Style-based feedback
Option to run the program multiple times
Technologies Used
Functions (def)
Conditional statements (if, elif, else)
Loops (while)
Variables
Return values (return)
String handling (.lower())
How to Run the Program
Open a terminal
Navigate to the folder where the file is located
Run the following command:
python outfit_rater.py
Example Usage

The user selects:

hoodie
svarta jeans
vita sneakers
casual

The program calculates a score and returns a result such as:
“7/10 – Snygg och välbalanserad outfit.”

Purpose of the Project

This project demonstrates the ability to:

structure a program using multiple functions
handle user input and process data
apply logical conditions in code
build a complete working application
